# Nexus Pulse – NCS Music Player

Nexus Pulse is a modern web-based music player built with HTML, CSS, and JavaScript.  
It features a futuristic silver and black interface and plays NCS tracks using the HTML5 Audio API.

---

## Features

- Playlist-based music playback
- Play, pause, next, and previous controls
- Shuffle, repeat, and autoplay modes
- Progress bar with seek support
- Volume control slider
- Interactive playlist
- Responsive user interface

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- HTML5 Audio API  
- Font Awesome

# Image Gallery – Silver & Black UI

A modern image gallery web application built using HTML, CSS, and JavaScript.  
It includes category filtering, lightbox viewing, and a premium silver and black design.

---

## Features

- Category-based image filtering
- Fullscreen lightbox view
- Previous and next navigation
- Keyboard support
- Responsive layout
- Live image counter

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Font Awesome
